Design Principles Summary

Remove

  • • All gradients (nav, modals, buttons, cards)
  • • All custom animations (liquid-glow, shimmer, bubble)
  • • Glassmorphism effects
  • • Multiple shadow depths
  • • Rainbow color coding
  • • Emoji as UI icons
  • • 140 lines of dead CSS

Add

  • • Design tokens (5 colors, 4 surfaces, 4 text sizes)
  • • Reusable Button component (4 variants)
  • • Reusable Card component (2 elevations)
  • • Reusable Modal wrapper
  • • Consistent border-radius (rounded-lg, rounded-xl)
  • • Inter font via Google Fonts
  • • More whitespace everywhere
This is a design mockup only — no changes have been made to the app.
